# Cloudflare R2

R2 is the default object storage for Overlay SaaS. It offers zero egress fees and S3-compatible APIs.

## Prerequisites

- [Cloudflare account](https://dash.cloudflare.com)
- R2 bucket created

## Setup

1. Go to **R2 > Manage R2 API Tokens**
2. Create a new token with **Object Read & Write** permissions
3. Note the **Access Key ID** and **Secret Access Key**

## Environment Variables

```bash
R2_ACCOUNT_ID=your-account-id
R2_ACCESS_KEY_ID=your-access-key
R2_SECRET_ACCESS_KEY=your-secret-key
R2_BUCKET_NAME=overlay-files
```

## Presigned URLs

Overlay automatically generates presigned URLs for uploads and downloads. Default TTL is 1 hour (configurable in `overlay.config.json`).

## CORS

Configure CORS on your R2 bucket to allow uploads from your domain:

```xml
<CORSConfiguration>
  <CORSRule>
    <AllowedOrigin>https://overlay.yourcompany.com</AllowedOrigin>
    <AllowedMethod>PUT</AllowedMethod>
    <AllowedMethod>GET</AllowedMethod>
    <AllowedHeader>*</AllowedHeader>
    <MaxAgeSeconds>3000</MaxAgeSeconds>
  </CORSRule>
</CORSConfiguration>
```
