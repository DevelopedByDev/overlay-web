# AWS S3

Overlay works with any S3-compatible provider: AWS S3, DigitalOcean Spaces, Wasabi, Backblaze B2.

## AWS S3 Setup

1. Create an IAM user with `s3:PutObject`, `s3:GetObject`, `s3:DeleteObject`
2. Note the **Access Key ID** and **Secret Access Key**

## Environment Variables

```bash
S3_ENDPOINT=https://s3.us-east-1.amazonaws.com
S3_ACCESS_KEY_ID=AKIA...
S3_SECRET_ACCESS_KEY=...
S3_BUCKET_NAME=overlay-files
S3_REGION=us-east-1
```

## DigitalOcean Spaces

```bash
S3_ENDPOINT=https://nyc3.digitaloceanspaces.com
S3_ACCESS_KEY_ID=...
S3_SECRET_ACCESS_KEY=...
S3_BUCKET_NAME=overlay-files
S3_REGION=nyc3
```

## CORS Configuration

```json
[
  {
    "AllowedHeaders": ["*"],
    "AllowedMethods": ["PUT", "GET", "DELETE"],
    "AllowedOrigins": ["https://overlay.yourcompany.com"],
    "MaxAgeSeconds": 3000
  }
]
```
