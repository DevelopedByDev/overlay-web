---
title: "Licensing"
description: "License boundaries for Overlay Web and related packages."
---

Overlay core product code is licensed under `AGPL-3.0-or-later`.

Reusable SDK and contract packages may use a more permissive license where noted in their package metadata. Check the package-level license fields before redistributing individual packages.

Modified public deployments and distributions must preserve license obligations and should not imply official Overlay branding unless LayerNorm Inc. grants written permission.

For repository-level terms, see:

- `LICENSE.md`
- `NOTICE.md`
- `TRADEMARKS.md`
- `docs/LICENSING.md`
