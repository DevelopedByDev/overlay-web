# Web Dead-Code Audit

Generated during Phase 5 mitigation.

## Method

- Static fan-in is generated by `npm run report:web-complexity`.
- Framework entrypoints (`src/app/**/page.tsx`, `loading.tsx`, `route.ts`, metadata image routes, middleware, instrumentation) are classified as `keep`.
- Package public API barrels are classified as `keep`.
- Tooling files can be allowlisted in `docs/reports/web-complexity-budgets.json`.
- Remaining zero-fan-in files are reviewed manually before deletion.
